HTML as One of My Core Skills

HTML is a markup language used to structure content on the web. It uses elements and tags to define sections, headings, paragraphs, links, images, lists, tables, forms, and more.

Examples of HTML Elements and Their Use

  • <h1> to <h6> - Headings. Use to define sections and sub-sections.
  • <p> - Paragraph. Use for text content.
  • <ul> <ol> <li> - Lists. Use for ordered or unordered lists.
  • <a> - Anchor/Link. Use to link to other pages or websites.
  • <img> - Image. Use to display pictures.
  • <table>, <tr>, <td> - Tables. Use to show tabular data.
  • <form>, <input>, <button> - Forms. Use to collect user input.
  • <section>, <header>, <footer> - Semantic structure. Use to organize content meaningfully.

Using HTML correctly ensures the website is easy to read, understand, and navigate. It is the foundation for all web development.
